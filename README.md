## Описание проекта
Проект представляет собой набор автоматизированных тестов для интернет-магазина OpenCart с использованием Selenium WebDriver. Тесты написаны в рамках итоговой работы по дисциплине «Модульное тестирование веб-приложений» для демонстрации навыков автоматизации тестирования веб-интерфейсов.

Основные технологии, применённые в проекте:
- **Selenium WebDriver** — для управления браузером и взаимодействия с веб-элементами
- **Pytest** — в качестве фреймворка для написания и запуска тестов
- **PageObject Pattern** — для организации кода и повышения его поддерживаемости
- **Allure** — для генерации наглядных отчётов о тестировании
- **Docker** — для развёртывания тестируемого окружения (OpenCart)
- **Git** — для контроля версий и хранения проекта

Тестируемое приложение: **OpenCart** — популярная платформа для интернет-торговли с открытым исходным кодом. Приложение запускается локально через Docker Compose.
